# Networking Module

A networking module that should:

- [x] Create a VPC with a given CIDR block.
- [x] Allow the user to provide the configuration for multiple subnets.
  - [x] Provide the subnet **CIDR blocks**.
  - [x] Provide the AWS **availability zone** for the subnet (validated against the region).
  - [ ] Mark a subnet as **public** or **private**.
